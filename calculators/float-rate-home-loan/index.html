How This Simulator Works

Each year you can do one action - either a part payment, a rate change, or both together. Click the Alter button on any month row to open the event modal.

Once an event is set for a year, all other months in that year are locked. The last 3 months of the loan are always disabled.

Note: This is an illustration tool. Actual bank charges, prepayment fees and GST are not modelled.
